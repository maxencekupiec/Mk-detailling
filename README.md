# MK Detailling

Site vitrine officiel de **MK Detailling** — Nettoyage auto professionnel à domicile à Nancy, Metz et au Luxembourg.

## 🚀 Lancer le site en local

Le site est en HTML/CSS/JS pur — aucune dépendance.

```bash
# Ouvrir directement dans le navigateur
open index.html

# Ou lancer un serveur local simple
python3 -m http.server 8080
# puis visiter http://localhost:8080
```

## 📁 Structure

```
site/
├── index.html              # Page principale
├── mentions-legales.html   # Mentions légales
├── confidentialite.html    # Politique de confidentialité
├── css/style.css           # Styles
├── js/main.js              # Animations et interactions
├── assets/                 # Images, vidéos, logo, carte France
│   └── galerie/            # Photos avant/après
└── favicon.ico
```

## 🌐 Déploiement

Recommandé : **Netlify** ou **Vercel** (drag & drop du dossier `site/`).
